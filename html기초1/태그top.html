많이 사용되는 태그

  • 목록을 표현하는 태그
  • 순서가 표시 안되는 태그
header - 페이지의 상단 영역 표현
footer - 페이지의 하단 영역 표현
main - 페이지의 본문, 주요 콘텐츠
nav - 메뉴
strong - 텍스트 강조(굵게)

입력태그 - 진짜 사용자들이 이용하는 사이트에 필수 태그

form - 입력 태그들의 폼, 틀
input - 입력할수 있는 태그

테이플 (표) 태그

table - 표 만들기 태그 tr- 테이블에서 행을 표현하는 태그 td - 테이블에서 텍스트를 작성할수 있는 태그
th - 테이블의 필드명 넣을수 있는 태그 (번호, 제목, 작성일 등등) thead - th가 있는 영역 tbody - td가 있는 영역
